What's happening

example-smart-app.js has been customized to include a crfurl variable populated with 'https://ocform3.mystudy.me/preview/::YYBD?'. This refers to an OC Participate form in preview mode on demo1. It then uses some (very cruddy) javascript to append to the URL a few of the variables pulled from the patient record. The GET parameters of the URL are constructed to map to the relevant OCP Form fields using this method. So, when you click the CRF link, the form opens with the values from the patient's health record pre-populated.
https://ocform3.mystudy.me/preview/::YYBD? <== the base form URL
d[/F_PHYSICALEXAM_ENGLISH/IG_PHYSI_UNGROUPED/I_PHYSI_SYSTOLIC]=systolicbp <== the systolic bp, where 'systolicbp' is replaced by the code with the actual value
&d[/F_PHYSICALEXAM_ENGLISH/IG_PHYSI_UNGROUPED/I_PHYSI_DIASTOLIC]=diastolicbp <== the diastolic bp, where 'diastolicbp' is replaced by the code with the actual value
